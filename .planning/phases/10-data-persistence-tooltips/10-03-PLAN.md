<objective>
Extend PackDataMessage with per-item register metadata and update pack drill-down renderers to set tooltip data attributes, delivering full D-15 compliance for pack drill-down views.

Purpose: Plan 02 delivers tooltips for streamed main sections but not for batch-rendered pack drill-down views. D-15 (locked) requires "every parameter value in the app" to have tooltips, including pack drill-down cell voltage/temperature values. This plan closes that gap by extending the backend PackDataMessage with per-item metadata and wiring the frontend pack renderers to set the same data attributes the tooltip system reads.

Output: Extended PackGroup with ItemMeta, populated by buildPackDataMessage, consumed by frontend pack renderers. Pack drill-down values get tooltips.
</objective>
