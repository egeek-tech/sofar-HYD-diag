```go
package register

import (
    "encoding/binary"
    "fmt"
    "strings"
)

// FormatValue interprets raw register bytes according to the probe's type configuration
// and returns a human-readable string. Returns the formatted value only (no prefix/address).
// Extracted from main.go.bak formatResult() lines 433-460.
func FormatValue(p Probe, data []byte) string {
    if p.IsASCII {
        return fmt.Sprintf("%q", strings.TrimRight(string(data), "\x00 "))
    }

    if len(data) < 2 {
        return "<no data>"
    }

    if p.Signed {
        val := int16(binary.BigEndian.Uint16(data[:2]))
        if p.Scale > 0 && p.Unit != "" {
            return fmt.Sprintf("%.2f %s", float64(val)*p.Scale, p.Unit)
        }
        return fmt.Sprintf("%d", val)
    }

    val := binary.BigEndian.Uint16(data[:2])
    if p.Scale > 0 && p.Unit != "" {
        return fmt.Sprintf("%.2f %s", float64(val)*p.Scale, p.Unit)
    }
    return fmt.Sprintf("%d (0x%04X)", val, val)
}
```

Define connection state type and events (per D-17):

```go
package broker

// State represents the broker's connection state.
type State int

const (
    StateDisconnected State = iota
    StateConnecting
    StateConnected
    StateReconnecting
)

// String returns a human-readable state name.
func (s State) String() string {
    switch s {
    case StateDisconnected:
        return "disconnected"
    case StateConnecting:
        return "connecting"
    case StateConnected:
        return "connected"
    case StateReconnecting:
        return "reconnecting"
    default:
        return "unknown"
    }
}

// StateEvent represents a connection state change.
type StateEvent struct {
    State State
    Err   error // non-nil if state change was caused by an error
}
```

**File: internal/broker/backoff.go**

Exponential backoff with cap (per research pattern, D-14):

```go
package broker

import "time"

// Backoff implements exponential backoff with a cap.
type Backoff struct {
    base    time.Duration
    max     time.Duration
    current time.Duration
}

// NewBackoff creates a backoff starting at base, capping at max.
func NewBackoff(base, max time.Duration) *Backoff {
    return &Backoff{base: base, max: max, current: base}
}

// Next returns the current delay and doubles it for next call (capped at max).
func (b *Backoff) Next() time.Duration {
    d := b.current
    b.current *= 2
    if b.current > b.max {
        b.current = b.max
    }
    return d
}

// Reset resets the backoff to the base duration.
func (b *Backoff) Reset() {
    b.current = b.base
}
```

- Create broker with buffered channels
- Backoff: base=1s, max=30s
- interReadDelay: 500ms
- Do NOT start the run goroutine in constructor -- provide a `Run(ctx context.Context) error` method

Public methods (all accept context.Context per D-16):
- `Run(ctx context.Context)` -- starts the command loop, blocks until ctx cancelled. This is called in a goroutine by the caller.
- `ReadRegisters(ctx context.Context, addr uint16, count uint16) ([]byte, error)` -- sends CmdRead, blocks on response
- `WriteRegister(ctx context.Context, addr uint16, value uint16) error` -- sends CmdWrite, blocks on response
- `ReadBatch(ctx context.Context, reads []ReadRequest) []Result` -- sends CmdReadBatch (per D-15), returns slice of results respecting inter-read delays
- `State() State` -- returns current state (read-only, safe without lock since only run goroutine writes)
- `StateEvents() <-chan StateEvent` -- returns read-only channel for state events (per D-17)
- `Address() string` -- returns configured inverter address
- `Close()` -- closes command channel

Internal methods (only called from run goroutine):
- `run(ctx context.Context)` -- select loop on ctx.Done and commands channel
- `execute(cmd command)` -- dispatch to executeRead or executeWrite
- `executeRead(req ReadRequest) Result` -- enforce inter-read delay, then call modbus read function
- `executeWrite(req WriteRequest) Result` -- call modbus write function
- `executeBatch(req BatchReadRequest) BatchResult` -- loop over reads with inter-read delay between each
- `ensureConnected() error` -- if conn is nil, attempt connect; on failure, enter reconnect loop with backoff
- `handleError(err error)` -- close conn, set state to Reconnecting, emit StateEvent
- `reconnect(ctx context.Context) error` -- loop with backoff delays until connected or ctx cancelled

The `ensureConnected` method uses `modbus.Connect(b.addr)` to establish the TCP connection. On success, reset backoff and emit `StateConnected`. On failure, emit `StateReconnecting` and backoff.

```go
elapsed := time.Since(b.lastReadTime)
if elapsed < b.interReadDelay {

}
```
