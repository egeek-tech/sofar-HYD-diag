**File 3: `internal/register/dcdc.go`**
Create DCDC section with ProbeGroups organized as follows. Read section 5.7 from the memory protocol reference for complete register details. Only include read-only (R) registers, skip RW/W control registers (per project constraint: read-only diagnostic tool).

Groups to create:
- "System Info" group: DCDC system info registers from 0x5004-0x5032
  - Include version/SN as ASCII registers with appropriate Count
  - 0x5004: first system info register (check memory ref for exact fields)
  - SN registers 0x5029-0x5032: ASCII, Count=10 (5 register pairs)
- "Real-Time Data" group: monitoring registers from 0x5044-0x5077
  - 0x5044: Parallel DCDC count (U16)
  - 0x5046: System running state (U16, Enum: DCDCRunningStateEnum)
  - 0x5047: Power limiting state (U16)
  - 0x5048: SOC balanced power (S16, Unit: "%", Scale: 0.1)
  - 0x504E: LV side bus voltage (S16, Unit: "V", Scale: 0.1)
  - 0x5050: LV side current (S16, Unit: "A", Scale: 0.1)
  - 0x5051: LV side power (S16, Unit: "kW", Scale: 0.01)
  - 0x505C: HV bus voltage (S16, Unit: "V", Scale: 0.1)
  - 0x505E: Insulation impedance (U16, Unit: "k\u03a9", Scale: 1)
  - 0x5060: Internal env temp (S16, Unit: "\u00b0C", Scale: 1)
  - 0x5061: Radiator temp 1 (S16, Unit: "\u00b0C", Scale: 1)
  - 0x5062: Radiator temp 2 (S16, Unit: "\u00b0C", Scale: 1)
- "Faults" group: fault/state registers 0x5065-0x506E
  - Fault 1-10 (0x5065-0x506E): each U16, Count: 1
  - Note: these are bitmap registers but we display them as raw values (no bitmap decoding for now, same as system fault display)
- "Capacity" group: charge/discharge capacity 0x5074-0x5077
  - 0x5074-0x5075: Total charge capacity (U32, Unit: "Wh", Scale: 1)
  - 0x5076-0x5077: Total discharge capacity (U32, Unit: "Wh", Scale: 1)

Export as `var DCDCGroups = []ProbeGroup{...}`

Read the full V1.38 memory reference to capture any additional DCDC real-time registers between 0x5044-0x5077 not listed above. The memory ref may have registers at intermediate addresses (0x5045, 0x5049-0x504D, 0x5052-0x505B, 0x505D, 0x505F, 0x5063-0x5064, 0x506F-0x5073). Include any that are documented as read-only monitoring registers.
