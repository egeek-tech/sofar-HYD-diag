**File 2: `tools/xlsx-discover/xlsx_parser.go`**
- First line: `//go:build xlsx_discover`
- Package `main`
- Import `github.com/xuri/excelize/v2`, `strconv`, `strings`, `fmt`
- Function `parseXLSX(path string) (map[uint16]*RegisterInfo, error)`:
  - Open file with `excelize.OpenFile(path)`
  - defer `f.Close()`
  - Get rows from "Address Description" sheet via `f.GetRows("Address Description")`
  - Skip row 0 (header)
  - For each row: check `len(row) >= 2` and address column (index 1)
  - Skip rows where address column contains " -- " (ranges) or is not exactly 4 hex chars
  - Skip rows where address is empty or column 2 (name) is empty
  - Parse address: `strconv.ParseUint(row[1], 16, 16)`
  - Extract metadata per RESEARCH.md column layout:
    - Col 1: address (hex 4-char)
    - Col 2: field name
    - Col 4: data type (U16, I16, U32, ASCII, BCD16, U64)
    - Col 5: scale/precision (take first line if multi-line via `strings.Split(s, "\n")[0]`)
    - Col 6: unit (handle CJK/Unicode by mapping known problematic strings)
    - Col 9: R/RW/W property
  - Map "I16" to "S16" for consistency (per Research Pitfall 5)
  - Return `map[uint16]*RegisterInfo` with `Source: "xlsx"`
- Helper `parseScale(s string) float64`: parse scale string, default to 0 if empty/invalid, handle multi-line
- Helper `cleanUnit(s string) string`: map known CJK unit strings to proper abbreviations

**File 1: `tools/xlsx-discover/comparison.go`**
- First line: `//go:build xlsx_discover`
- Package `main`
- Import `fmt`, `os`, `sort`, `text/tabwriter`

Define:
```go
type ComparisonEntry struct {
    Addr   uint16
    XLSX   *RegisterInfo // nil if not in XLSX
    V138   *RegisterInfo // nil if not in V1.38
    Probes *RegisterInfo // nil if not in current probes
}
```

Function `compare(xlsx, v138, probes map[uint16]*RegisterInfo) []ComparisonEntry`:
- Create a set of all unique addresses from all three maps
- For each address, create a ComparisonEntry with pointers from each map (nil if absent)
- Sort by address ascending
- Return sorted slice

Function `statusLabel(e ComparisonEntry) string`:
- If all three non-nil: return "all three"
- If xlsx+v138 non-nil, probes nil: return "missing from probes"
- If xlsx only: return "XLSX only"
- If v138 only: return "V1.38 only"
- If probes only: return "probes only"
- If xlsx+probes, no v138: return "XLSX+probes"
- If v138+probes, no xlsx: return "V1.38+probes"
- Other combinations labeled accordingly

Function `printComparison(entries []ComparisonEntry)`:
- Group entries by address range using these groups (per D-03):
  - "General (0x00xx)" for 0x0000-0x00FF
  - "System (0x04xx)" for 0x0400-0x04FF
  - "EPS/PV (0x05xx)" for 0x0500-0x05FF
  - "Battery (0x06xx)" for 0x0600-0x06FF
  - "Arc (0x07xx)" for 0x0700-0x707F
  - "Meter (0x70xx)" for 0x7080-0x70FF
  - "Safety (0x08xx-0x0Axx)" for 0x0800-0x0AFF
  - "Config (0x10xx-0x13xx)" for 0x1000-0x13FF
  - "Network/Control (0x20xx)" for 0x2000-0x20FF
  - "Compatible (0x4Exx)" for 0x4E00-0x4EFF
  - "DCDC (0x50xx-0x54xx)" for 0x5000-0x54FF
  - "PCU (0x60xx)" for 0x6000-0x607F
  - "BDU (0x60xx)" for 0x6080-0x60FF
  - "BMS (0x90xx-0x91xx)" for 0x9000-0x91FF
- For each group, print a section header: `=== {GroupName} ===`
- Use `text/tabwriter` (per Research stdlib recommendation) to align columns
- Columns: ADDR, NAME, XLSX, V1.38, PROBES, STATUS
  - ADDR: `0x%04X` format
  - NAME: from first non-nil source, truncated to 30 chars
  - XLSX/V1.38/PROBES: "yes" or "-"
  - STATUS: from `statusLabel()`
- At end of output, print summary: total registers per source, total unique, count of "missing from probes"

Per D-05, function `currentProbeAddrs() map[uint16]*RegisterInfo`:
- Import `sofar-hyd-diag/internal/register` (always in sync with code)
- Collect from all exported group variables:
  - `register.SystemGroups`
  - `register.GridGroups`
  - `register.EPSGroups`
  - `register.GeneratePVGroups(16)` (max channels to capture all possible PV addresses)
  - `register.GenerateBatteryGroups(8)` (max channels)
  - `register.ConfigurationGroups`
  - `register.StatisticsGroups()`
  - `register.BMSInfoGroups()`
  - `register.FaultRegisters`
- Also collect from flat probe functions:
  - `register.PackRTProbes()`
  - `register.PackInfoProbes()`
  - `register.PackTemps58Probes()`
  - `register.PackProbeGroups()`
- For each probe, create a `RegisterInfo` with Addr, Name, and derive Type from probe fields:
  - If `p.IsASCII` -> Type="ASCII"
  - If `p.U32` -> Type="U32"
  - If `p.Signed` -> Type="S16"
  - Else -> Type="U16"
  - Copy Scale, Unit from probe
  - Source="probes"
- Return map keyed by address
