  **Part B: Implement batch accumulation (GREEN)**

  In `internal/hub/hub_streaming.go`, modify `streamPackRead` Step 4 (the inner loop starting around line 325). Replace the current pattern of sending each result immediately with per-group accumulation:

  Change from:
  ```go
  // Step 4: Read each probe individually, streaming register_value messages
  for _, g := range groups {
      for _, p := range g.Probes {
          if readCtx.Err() != nil {
              return
          }
          if skipRegs[p.Addr] {
              continue
          }
          data, err := h.broker.ReadRegisters(readCtx, p.Addr, p.Count)
          if err != nil {
              // ... error handling ...
              h.results <- sectionResult{
                  section: "bms",
                  msg:     NewRegisterValue("bms", g.Name, p.Name, "", errStr, p.Addr, ""),
              }
              continue
          }
          if readCtx.Err() != nil {
              return
          }
          h.results <- sectionResult{
              section: "bms",
              msg:     NewRegisterValue("bms", g.Name, p.Name, FormatValue(p, data), "", p.Addr, FormatRawValue(p, data)),
          }
      }
  }
  ```

  To:
  ```go
  // Step 4: Read probes per group, accumulate results, send as batch (D-05)
  for _, g := range groups {
      var groupResults []sectionResult
      for _, p := range g.Probes {
          if readCtx.Err() != nil {
              return
          }
          if skipRegs[p.Addr] {
              continue
          }
          data, err := h.broker.ReadRegisters(readCtx, p.Addr, p.Count)
          if err != nil {
              errStr := err.Error()
              if strings.Contains(errStr, "timeout") || strings.Contains(errStr, "0x02") || strings.Contains(errStr, "illegal") {
                  skipRegs[p.Addr] = true
              }
              if readCtx.Err() != nil {
                  return
              }
              groupResults = append(groupResults, sectionResult{
                  section: "bms",
                  msg:     NewRegisterValue("bms", g.Name, p.Name, "", errStr, p.Addr, ""),
              })
              continue
          }
          if readCtx.Err() != nil {
              return
          }
          groupResults = append(groupResults, sectionResult{
              section: "bms",
              msg:     NewRegisterValue("bms", g.Name, p.Name, FormatValue(p, data), "", p.Addr, FormatRawValue(p, data)),
          })
      }
      // D-05: Send all results for this group at once
      for _, r := range groupResults {
          h.results <- r
      }
  }
  ```

  Key changes:
  - Replace `h.results <- sectionResult{...}` inside the inner probe loop with `groupResults = append(groupResults, sectionResult{...})`
  - After the inner probe loop completes for a group, iterate `groupResults` and send each to `h.results`
  - Move the error handling (skipRegs tracking) to accumulate instead of send
  - The context cancellation check (`readCtx.Err()`) remains in the inner loop for early exit

  Update the comment at line 278 to reflect the new behavior:

  ```go
  // streamPackRead reads pack registers group-by-group, accumulating results per group
  // and sending them as a batch when each group completes. This creates a per-group
  // streaming effect where all values in a group appear together in the UI (D-05, D-06).
  ```
