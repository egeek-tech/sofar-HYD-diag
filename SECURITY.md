# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities privately via
[GitHub Security Advisories](https://github.com/egeek-tech/sofar-HYD-diag/security/advisories/new).

Do not open public issues or pull requests for security problems.

You should receive an acknowledgement within a few days. After triage,
a fix and coordinated disclosure timeline will be discussed in the advisory.

## Scope

This policy applies to the latest released version on the `master` branch
and the most recent published Docker image and release artifacts. Older
versions are not supported.
